    hash_password() click to toggle source

    Set password to a hashed version of the password given by entry form if the password field is not blank.

    # File app/models/extensions/passworded.rb, line 36
def hash_password
  if(!self.password_plaintext.blank?)
    self.password = Extensions::Passworded.hash_password(self.password_plaintext)
  end
end
    password_validation_required?() click to toggle source

    Check if the entry form requires password field. Password validation is only required if this is a new Passworded item or if the user has entered something into the password field.

    # File app/models/extensions/passworded.rb, line 44
def password_validation_required?
  self.new? ? true : !self.password_plaintext.blank?
end

def valid_password?(password_plaintext)
  self.password == Extensions::Passworded.hash_password(password_plaintext)
